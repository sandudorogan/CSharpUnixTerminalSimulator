# Unix Terminal Simulator
======
## C#
------
..As the name implies it, the repository contains a *simplist* terminal simulator which supports the abridged versions of the following commands:
* `Echo` message
* `Echo` message > file
* `Cd`
* `Ls`
* `Touch`
* `Mkdir`
* `Tree`
* `Pwd`
* `Rm`
* `Rmdir`
